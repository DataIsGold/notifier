Notifier

Lindas notificações, com CSS e Javascript puro!

Trabalhando em um projeto e precisando de uma solução para notificações? Boa notícia \o/
O notifier pode te ajudar com essa tarefa, basta importar para o seu projeto e começar a usar!

Notificações Padrão

        
          notifier.show('Default!', 'Sou uma notificação Default.', '', '', 0);
          notifier.show('Default!', 'Você tem uma reunião agendada às 10:30h.', '', '', 0);
          notifier.show('Default!', 'Você acabou de cadastrar seu currículo com sucesso.', '', '', 0);
          notifier.show('Default!', 'Os dados apresentados nessa tela podem sofrer alterações.', '', '', 0);
          notifier.show('Default!', 'Ocorreu um erro! Não foi possível salvar o registro.', '', '', 0);
