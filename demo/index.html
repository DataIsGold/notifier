Notifier

Lindas notificações, com CSS e Javascript puro!

Trabalhando em um projeto e precisando de uma solução para notificações? Boa notícia \o/
O notifier pode te ajudar com essa tarefa! Basta importar para o seu projeto e começar a usar!

Download ZIP Projeto GitHub

Como Usar?

Muito Simples!

  1. Baixe o projeto.
  2. Exporte os arquivos.
  3. Copie os arquivos "css/notifier.css" e "js/notifier.css" para dentro do seu projeto.
  4. Importe o CSS no <header>
  5. Importe o JS antes do fechamento do <body>
  6. Agora é só ser feliz! ;)

Notificações Padrão

        
          notifier.show('Default!' , 'Sou uma notificação Default.', '', '', 0);
          notifier.show('Lembrete!', 'Você tem uma reunião agendada às 10:30h.', '', '', 0);
          notifier.show('Parabéns!', 'Você acabou de cadastrar seu currículo com sucesso.', '', '', 0);
          notifier.show('Atenção!' , 'Os dados apresentados nessa tela podem sofrer alterações.', '', '', 0);
          notifier.show('Desculpe!', 'Ocorreu um erro! Não foi possível salvar o registro.', '', '', 0);
